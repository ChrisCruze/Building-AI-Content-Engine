Cuculi-AI-Marketing: A production repository where structure dictates system logic

The system leverages a file-based architecture to orchestrate multi-agent workflows, utilizing Airtable as the source of truth and MongoDB for data enrichment to drive automated content generation.

1. Integration Foundation

Airtable (Orchestrator)

Acts as the "Single Source of Truth." Manages Idea entry, Status tracking, and Metadata (Keywords, Audience). Triggers the pipeline via run.py.

MongoDB (Context)

Enriches content with real-world user data (e.g., "Never-Engaged Veterans") and event statistics to ground AI generation in reality.

Web Interface (index.html)

Pure frontend UI for human-in-the-loop review. Allows editing, status updates, and final publishing approval.
