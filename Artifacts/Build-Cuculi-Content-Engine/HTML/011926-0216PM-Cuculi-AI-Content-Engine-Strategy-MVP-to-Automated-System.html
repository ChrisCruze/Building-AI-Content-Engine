1. Strategic Reality

Core Philosophy

  • Speed First: Deliver value immediately via manual execution.
  • System Second: Layer automation only after the process is proven.
  • Compounding Work: Every manual brief and template becomes future automation logic.

1. Idea (Airtable) →
2. Brief (Template) →
3. Outline (Structure) →
4. Research (Data) →
5. Writer (Draft) →
6. Editor (Refine) →
7. Output (Google Doc)

Pillar A: Integrations

Airtable: Control center & source of truth.
MongoDB: Event data for content personalization.
LLMs: Claude (Writing) & ChatGPT (Logic).

Pillar B: Intelligent Agents

Agent Role Primary Function
Planner Strategy & Audience
Researcher Signals & Data

Writer/Editor Drafting & Voice

  • NO Integration / NO Code
  • Defined Templates (`brief.md`)
  • Human Execution of Logic
PHASE 2: FULL SYSTEM

Timeline: Weeks/Months

Focus: Scale & Automation.

  • Full Airtable Orchestration
  • MongoDB Data Enrichment
  • Agents running autonomously
